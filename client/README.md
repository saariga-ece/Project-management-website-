# JiraLite Frontend

This is the React + Vite frontend for the Jira-inspired project management application.

## Run the frontend

1. Install dependencies:
   ```bash
   cd client
   npm install
   ```

2. Start the app:
   ```bash
   npm run dev
   ```

3. Open the browser at `http://localhost:5173`.

## Features

- Dashboard overview with project and task metrics
- Board view with Kanban columns by task status
- Create, edit, move, and delete tasks
- Create and delete projects

## API

The frontend communicates with the backend API at `http://localhost:4000/api`.
